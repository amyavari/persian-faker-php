## slug

یک slug تصادفی تولید می کند. تعداد کلمات داخل slug را می توان با پارامتر `$nbWords` تعیین کرد (مقدار پیش فرض `6` است). تعداد کلمات یک slug می تواند تغییر کند یا شما می توانید مقدار پارامتر `$variableNbWords` را برابر `false` قرار دهید تا همیشه همین تعداد کلمه داخل slug باشد.

```php
echo $persianFaker->slug();
// 'نگاه-تصویر-برق-اتوبوس', 'کلاس-صادق-کاغذ-انگور-ساز-روز'

echo $persianFaker->slug(nbWords: 4);
// 'خیانت-شفق-سکوت-گیاه-آشپزخانه', 'دکتر-سبزی-مادیات'

echo $persianFaker->slug(nbWords: 4, variableNbWords: false);
// 'نگاه-تصویر-برق-اتوبوس' ,'خیانت-شفق-سکوت-گیاه'
```
