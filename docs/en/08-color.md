# 08 Color

## safeColorName

Generates a random color name from a set of 16 primary color names.

```php
echo $persianFaker->safeColorName();
// 'سیاه', 'آبی', 'زرد'
```

## colorName

Generates a random color name.

```php
echo $persianFaker->colorName();
// 'سیاه', 'آبی', 'زرد', 'سبز زمردی', 'زغالی'
```
